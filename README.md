# HTML, CSS & JavaScript Projects

A collection of mini frontend projects built using **HTML**, **CSS**, and **JavaScript**. This repository showcases my learning journey through hands-on practice by building real-world UI components, utilities, and interactive web applications.

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)

## 📂 Projects

| Project                   | Description                             |
| ------------------------- | --------------------------------------- |
| Calculator                | Perform basic arithmetic operations     |
| BMI Calculator            | Calculate Body Mass Index               |
| Loan Calculator           | Calculate monthly loan payments         |
| Accordion                 | Expand and collapse content sections    |
| Counter App               | Increment, decrement, and reset counter |
| To-Do List                | Manage daily tasks                      |
| Password Strength Checker | Check password strength                 |
| Character Counter         | Counts character at real-time           |
| Tip Calculator            | Calculates the tip using percentage     |

> More projects will be added regularly as I continue learning.
